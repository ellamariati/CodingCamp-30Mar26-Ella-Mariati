# Expense & Budget Visualizer

Aplikasi web untuk mencatat dan memvisualisasikan pengeluaran harian.

## Fitur
- Input transaksi (nama, jumlah, kategori)
- Daftar transaksi dengan tombol hapus
- Total balance otomatis
- Pie chart pengeluaran per kategori
- Kategori kustom
- Ringkasan bulanan
- Sort transaksi
- Highlight pengeluaran melebihi limit
- Dark/light mode toggle

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript
- Chart.js
- Local Storage
